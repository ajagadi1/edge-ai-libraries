Build Steps 

1. Download VS Build Tools  (Seach for this -> Build Tools for Visual Studio 2026)
	https://visualstudio.microsoft.com/downloads/#build-tools-for-visual-studio-2026
	Choose these Individual Components from 2nd Tab (can be further review and reduce if possible)
	1. vcpkg package manager
	2. Visual C++ v14 Redistributable Update
	3. MSVC Build Tools for x64/x86 (Latest)
	4. Windows 11 SDK (10.0.26100.7705)
2. Make sure if you have installd below for windows
	1. cmake for windows
	2. git for windows
	3. powershell for windows (normally available) >7.5.0
3. Download windows dll resources of DLStreamer from here
	https://github.com/open-edge-platform/dlstreamer/releases/download/v2026.0.0/dlstreamer_dlls_2026.0.0.zip
	
	Unzip it.
	
4. Launch powershell as administrator and run setup_dls_env.ps1  from DLStreamer resurces. (This would download all dependencies that are required to run DLS DLLs e.g. gstreamers etc.) 

5. Download src-gst-gencamsrc.zip to any folder on widnows and run build_gencamsrc.ps1 to build gencamsrc plugin. Run below command from root folder of this zip
	.\build_gencamsrc.ps1 -GenicamRoot "./plugins/genicam-core/genicam_win/" -VcVersion 120 
6. Plugin should get build and generated into this folder  ~/build/bin/Release/gstgencamsrc.dll . Copy it to C:\dlstreamer_dlls\

Building plugin on linux 
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build -j$(nproc)
sudo cmake --install build


Run Steps 
1.install camera sdk that is necessary (pylon/balluff/hikvision)

1.1 instead of building gencamsrc, Download windows dll resources of DLStreamer from here
	https://github.com/open-edge-platform/dlstreamer/releases/download/v2026.0.0/dlstreamer_dlls_2026.0.0.zip and Unzip it. Launch powershell as administrator and run setup_dls_env.ps1  from DLStreamer resurces. (This would download all dependencies that are required to run DLS DLLs e.g. gstreamers etc.) copy gstgencamsrc.dll to C:\dlstreamer_dlls\

2. Set below env as per your installation folder paths.  ($vc120, $GENICAM_GENTL64_PATH can vary as per your system installation, below it points to all possible values for Basler, Ballus and HikRobot camera, change accoridnging)

$vc120 = "C:\Users\intel\edge-ai-libraries-main\microservices\dlstreamer-pipeline-server\plugins\camera\src-gst-gencamsrc\plugins\genicam-core\genicam_win\Runtime\bin\Win64_x64"
$gstRoot = "C:\Program Files\gstreamer\1.0\msvc_x86_64"
$dls = "C:\dlstreamer_dlls"
$env:PATH = "$vc120;$gstRoot\bin;$dls;" + $env:PATH
$env:GST_PLUGIN_PATH = $dls
$env:GST_PLUGIN_SYSTEM_PATH_1_0 = "$gstRoot\lib\gstreamer-1.0"
$env:GENICAM_GENTL64_PATH = "C:\Program Files\Basler\pylon\Runtime\x64;C:\Program Files\Basler\pylon\Runtime\x64\pylonCXP\bin;C:\Program Files (x86)\Common Files\MVS\Runtime\Win64_x64;C:\Program Files\Balluff\ImpactAcquire\bin\x64"
$env:GST_REGISTRY_1_0 = "C:\Temp\gst-registry-clean.bin"
gst-inspect-1.0 gencamsrc
